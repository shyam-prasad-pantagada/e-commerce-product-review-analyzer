ℹ️ About This Project

This E-commerce Product Sentiment Analysis system analyzes customer reviews to determine sentiment (Positive, Negative, Neutral) based on ratings (1-5 scale).

Note: Ratings are mapped as follows:
⭐⭐⭐⭐⭐ (5), ⭐⭐⭐⭐ (4) → Positive
⭐⭐⭐ (3) → Neutral
⭐ (1), ⭐⭐ (2) → Negative

Features:

  • 📋 Product Selection: Dropdown list of all products from dataset
  • 🔍 Sentiment Analysis: Analyze reviews for selected product
  • 📈 Visualization: Bar charts showing sentiment distribution
  • 📊 Real-time Statistics: Overall and product-specific stats
  • 🛍️ Product Browser: View all products in dataset
  • 📝 Review Display: See actual reviews with sentiment labels

How to Use:

  1. Go to Product Analysis tab
  2. Select a product from dropdown or search
  3. Click "Analyze Sentiment" button
  4. View sentiment results and reviews
  5. Click "Show Visualization" for charts
  6. Go to Visualization tab for more chart options
